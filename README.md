# BMS Maps Data Converter

Extracts station data from Falcon BMS data files and stores it in a JSON data structure needed for the BMS Maps website.

## Balkan 

### Countries Codes

* LI = Italy
* LD = Croatia
* LY = Serbia and Montenegro
* LQ = Bosnia and Herzegovina
* LG = Greece
* LJ = Slovenia
* LH = Hungary
* LA = Albania
* LW = North Macedonia
* BK = Kosovo
